由于早期缺乏对JavaScript系统的学习, 导致了我在很多基础的东西了解的不够深, 这份笔记已经初步完成了对JavaScript整个体系中最重要的一部分的梳理, 很大程度地弥补了我早期由于学习习惯不好留下的问题.
> 本笔记目前已经全部完成, 但是我需要整理成脑图,且为保证内容正确性和完整性, 我将会边检查边更新.
